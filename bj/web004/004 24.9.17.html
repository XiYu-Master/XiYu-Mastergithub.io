1.表格

1）作用
存储数据，向用户展示数据
2）标签 table>tr*3>td*3 如下是三行三列的表格
标题 caption
th thead 1
tbody 2 2
tfoot 3 3
3）table属性（表格）
  • border 边框
  • width 表格宽度
  • height 表格高度
  • align 表格对齐方式
  • bgcolor 背景颜色
  • background 背景图片
  • bordercolor 边框颜色
  • cellspacing 单元格之间的距离
  • cellpadding 内容距边框的距离

4）tr属性（行）
  • height 高度
  • align 行内容的水平对齐方式
  • valign 行内容的垂直对齐方式 （默认middle 居中|top 上|bottom 下）
  • bgcolor 背景颜色
  • background 背景图片

5）td属性（列/单元格）
  • 特点：同行等高 同列等宽
    即给一行的td单元格设置了高度，那这一行td的高都变
    给一列的td单元格设置了宽度，则一列td的宽都变
  • width 宽度
  • height 高度
  • align 单元格内容水平对齐方式
  • valign 单元格内容垂直对齐方式（默认middle 居中|top 上|bottom 下）
  • bgcolog 背景颜色
  • background 背景图片
  • 掌握
  • colspan 水平合并单元格
  • rowspan 垂直合并单元格

2.表单

1）作用
提交数据，使页面具有交互性
2）标签
  • form （一个页面可以有多个form表单 但form之间 不能互相嵌套）
  • 下面的皆写在form标签里面（掌握）
    • name 表单的名称
    • action 提交的地址
    • action 提交的方式方法 默认get 可取post
      • get和post的区别：
        1. get提交数据不安全 会在地址栏中显示 post则不会显示
        2. get只能提交少量数据，post理论上没有限制
3）表单元素

  • 前提：下列全写在form表单中

  • input type="text" 文本框
  • input type="password" 密码框
  • input type="radio" name="" 单选按钮（必须加上name属性才能多选一，鄙人自认为是把选项放到一道题的样子）
  • input type="checkbox" 多选按钮
  • input type="submit" 提交按钮
  • input type="reset" 重置按钮
  • input type="button" 空按钮
  • 《button》提交按钮《/button》
  • 《button type="reset"》重置《/button》
  • 《button type=“button”》空按钮《/button》
  • input type="file" 提交文件按钮 但默认表单不支持提交文件，需要修改form的编码格式 需要在标签内加上“enctype="multipart/form-data"
  • input type="hidden" 隐藏域 在页面中不显示 但携带顶点数据可以正常提交 （不想给用户看，又像把数据传过去的情况用）
  • input type="image" src=“” 图片提交按钮（sec为图片路径，必须有 ）
    • 下拉列表
  • 《select》
    《option value=“”》value（当前值）属性值为显示内容《/option》
    《option value=“”》《/option》
    《option value=“”》《/option》
    《/select》
    • 多行文本框
  • 《textarea name=“” id=“” cols=“” ros=“”》cols列 改变水平长度 rows行 改变垂直长度《/textarea》
    • label属性 -提升用户体验度
  • 《label for=“”》for属性和id属性绑定 属性一致《/label》应用场景为：点击描述按钮的字也能选上按钮关于表单我已经两个月没碰了，故不在此展示
    如-请选择您的性别：
  • 关于起名：value没有限制。id/name包含数字英文 且不能数字为开头
4）表单元素的属性
  • type 类型
  • name 名称
  • value 当前值
  • checked 默认被选中 配合单选按钮和多选按钮使用（唯一需另外掌握 上边仨基本都有）
  • selected 默认显示 配合option使用
  • disabled 禁用 只能看 不能编辑
  • readonly 只读
  • size 显示的数量 配合select使用 取值为number（数字）

07 -H5新增type类型

同样的，都写在form表单里
  • 邮箱：《input type="emall"》js判断只包含@几后面一个字符（任意字符），后面就没了。可以加name属性于地址栏（get情况下）显示提交数据{原话：必须包含@，@后面必须有内容}
  • 网址：《input type=“URL”》js判断只到http：//(任意内容，但必须有){原话：必须包含协议 协议后面必须有内容}
  • 搜索框：《input type=“search”》可见最后有×，后期可以改样式
  • 数字：《input type=“number”》（新增属性）可以设置min最小值 max最大值 step步长（但数字不能为其数任意倍数以外的数） （旧）value设置当前值
  • 范围（滑块）：《input type=“range”》 可加min max step value。
  • 颜色：《input type=“color”》可设value的16进制和RGB颜色 默认黑色，不支持颜色英文设置
  • 电话：《input type=“tel”》电脑端显示为文本框手机未知
  • 日期（天）：《input type=“date”》
  • 周：《input type=“week”》
  • 月：《input type=“month”》

1.DOCTYPE:document type 文档类型声明（不是html的标签） 作用：告诉浏览器按当前标准解析代码
2.图片 img（图片） src（ 必须属性：路径） alt（当图片不能正常显示时提示文本） title（鼠标悬停显示） height（高度） width（宽度）宽高等比例缩放，单位默认为像素（px）

2.路径

  • 绝对路径
    • 网址
    • 盘符
  • 相对路径
    • 平级 （直接写路径）
    • 下一级 /
    • 上一级 ../（每上一级就加一个../）

3.超链接

a .(文本，段落，图片等). /a
    属性
  • href（跳转路径，空链接时属性值为“#”）
  • target（设置打开窗口，默认_self【当前窗口打开】_blank【可以在新窗口打开】）
  • name（作锚点用）可以
    1点击a到a（注意在href属性值里先加一个#号，再输入想要跳转锚点的锚点的name属性值）
  • 2.跳转到块级元素
    用法： 给块级元素加一个id属性，在a的href属性值输入#加块级元素id属性值即可

4.块级元素和行内元素的区别

  • 1块级元素独占一行，行内元素在同一行显示
  • 2块级元素默认宽度为100% 行内元素默认宽度由内容撑开

5.文本格式化标签

双标记、行内元素。
  • b加粗
  • i倾斜
  • u下划线
  • s或del删除线
  • em强调（倾斜）
  • strong强调（加粗）
  • sup上标
  • sub下标（p的圈在下就是下标，在上就是上标）
  • big大号字体
  • small小号字体

6.列表

双标记 ，块级元素
  • 1无序列表
    • 1。一般结构：ul>li
    • 2。常用结构：ul>li>a
    • 3。嵌套：ul>li>ul>li
    • 4。ul和li的属性
      type：设置项目符号的项目类型
      取值
      disc 默认值：黑色实心圆
      circle 空心圆
      square 黑色实心矩形
      none 不显示
  • 2有序列表
    • 1基础结构：ol>li
    • 2常用结构:ol>li>a
    • 3嵌套:ol>li>ol>li 或 ol>li>ul>li
    • 4 ol和li的属性
      type：设置项目符号，默认1 可取值a A i I
      start：设置从第几个开始 取值数字
      reversed：倒序 reversed=“reversed”当属性等于属性值时，可以简写为属性
  • 3自定义列表（都是双标签）
    • dl自定义列表开始标签
    • dt自定义列表项 主体标签
    • dd自定义列表项 注释（描述）标签

7，表格

数几行几列时，table里有几个tr就有几行，每tr里有几个td/yh就有几列
  • 1作用：存储数据，向用户展示数据
  • 2标签：
    • table表格>tr行>td单元格（列）
      td可以换成th显示效果为单元格内水平居中且加粗
    • caption 表格的标题

03表格的标签（掌握）

都是双标签 一个table有几个tr就有几行。一个tr有几个td/th就有几列（横行竖列）
  • table 表格开始
  • tr 行
  • td 列，单元格
  • th 特殊单元格 内容默认水平居中且加粗
  • caption 表格的标题（居中显示）写在table和tr之间，可以嵌套文本格式化标签，如b，i，u
  • ↓ 新的（对样式没影响，只是划分表格区域）
  • thead 表格的头部 （位置要包含在tr外边）
  • tbody 表格的身体（浏览器默认普通单元格为身体）
  • tfoot 表格的脚部

04 table的属性（了解）

border用来设置表格边框尺寸的大小
align 设置表格对于网页的水平对齐方式 写在table标签里 默认居左。内容需要给tr（行）或者td（列）设置
cell（单元格） padding（内边距） 内容距边框的距离（是整个内边距为20，不是某一方向）
cellspacing（space，空间，距离）设置单元格(边框)之间的距离
width 设置单元格格宽度
height 设置单元格高度
bgcolor 表格的背景颜色
background 表格的背景图片（图片太小会自动复制填充整个表格）
bordercolor 表格的边框颜色 默认没有边框

05 tr（行）的属性（了解）

ps：和table属性一样，多了valign垂直对齐方式
height 设置行的高度
width 设置行的宽度（不超过单元格整体）
align 设置行内容的水平对齐方式，默认left 可以取值center\right
valign 设置行内容的垂直对齐方式 默认（middle）居中，可以取值top（顶）\bottom（底）
bgcolor 设置行的背景颜色
background 设置行的背景图片

06 td的属性（了解）

th也可以用 表格特点：同列等宽 同行等高
width 设置单元格的宽度（一列跟着改变）
height 设置单元格的高度（一行跟着改变）
align 设置单元格内容的水平对齐方式
valign 设置单元格内容的垂直对齐方式 默认middle居中 可以取值top顶 bottom底
bgcolor 设置单元格的背景颜色
background 设置单元格的背景图片

07合并单元格（掌握）

td的属性
colspan 水平合并，取值为number
属性写在从左往右开始的td里面
水平合并 删除同一行（一个tr标签里）的td
删除个数=合并个数-1
rowspan 垂直合并。取值为number（数字）
属性写在从上往下开始的td里面
垂直合并 删除下面几行（合并一行就删每下一层里的一个td）的td
删除的个数=合并的个数-1
教学用的是整体表格自己合并完了再删除，可以逆运用，但确实逆推理太费时

11form表单（掌握）

1作用
提交数据，使页面具有交互性（一阶段只写样式即可，三阶段才有服务器）
2标签
form 双标记，块级元素。里边写表单或想向服务器提交的元素（之前学标签的都可以套）用来提交数据（交互必备）
注意
一个页面可以有多个form
form之间不可以相互嵌套
属性（掌握）
name 表单的名称
action 提交的地址
method 提交的方式方法 默认get 可取值post
get和post的区别
（1）get提交的数据不安全，数据会在地址栏中显示；post不会显示
（2） get提交的数据大小有限制：post理论上没有限制
12 表单元素（表单域和表单按钮的共名）1（掌握）
一个完整的表单包含三个基本组成部分：表单标签（form），表单域（如账户 密码框），表单按钮（如点击登录）。
表单域和表单按钮都属于表单元素
(3)表单元素（除了form是块级，其他都是行级元素）
input （单标记）
属性
  • type 区分 input的类型
  • name （写在input就是这个input的名字，写在form就是整个form的名字）（引入驼峰命名法：前一个单词首字母小写 后一个单词首字母大写）
  • value 当前值（一般给系统看）
  • id 用来做联动例如和label标签联动做文字也能当选项，提升用户体验
input的type属性：
  • text 单行文本框
  • password 密码框
  • radio 单选按钮 注：必须加name属性（语法：input type=“radio” name=* 《—两个），而且名字保持一样才能达成单选效果，复选也一样，个人理解是归为同一个选择题目下。题号就相当于name
  • checkbox 复选框（复选按钮）
  • 按钮（行级元素）
  • submit 提交按钮 value属性值可以改变显示文字
  • reset 重置按钮 value属性值可以改变显示文字
  • button 空按钮 value可以更改显示文字
  • 下列三个不再用input做标签
  • button 双标记 功能：提交（由此得默认type属性值=“submit”）
    格式：《button》内容《button》
  • button type =“reset” 功能：重置
  • button type=“button” 功能：空按钮
